# 🚀 Galactic Horizons - Space Travel Agency

Welcome to the **Galactic Horizons** repository, a futuristic landing page for a space travel agency operating within the Solar System. This page serves as the main entry point for a space flight booking web application.

The project features a modern design, smooth animations, and is fully optimized for performance.

---

## 🛠️ Tech Stack

This project leverages the most modern web technologies to ensure top performance, excellent DX (Developer Experience), and a visually stunning user interface:

- **React 19** – A core JavaScript library for building component-based user interfaces.
- **Next.js 15 (App Router)** – A React framework for server-side rendering (SSR) and performance optimization.
- **TypeScript** – For a robust, strongly-typed codebase that eliminates runtime errors.
- **Tailwind CSS** – A utility-first CSS framework for rapid and highly customizable styling.
- **Framer Motion** – For advanced animations and smooth fade transitions (e.g., testimonial carousel).
- **Lucide React** – A clean and modern icon set tailored for the UI.

---

## ✨ Features

- **Space Hero Banner**: A high-impact hero section with a clear Call to Action (CTA) for user registration.
- **Dark & Light Mode Support**: 
  - *Light Mode*: Crisp, light background with subtle "stardust" gradients.
  - *Dark Mode*: Deep space-inspired tones enhanced by vibrant **Neon** accents (cyan, magenta, lime green).
- **Planet Exploration Grid**: Interactive planet cards featuring the main tourist attractions for each destination in the Solar System.
- **Testimonial Carousel**: A fading carousel displaying reviews, ratings, and avatars from our satisfied space travelers.
- **Safety & Trust Sections**: Highlighting the agency's advanced safety protocols and the key reasons to choose our fleet.
- **Smooth Scrolling & Animations**: Dynamic fade-up effects triggered as the user scrolls down the page.

## 🚀 Installation & Local Development

Follow these steps to clone and run the project locally on your machine:

### 1. Clone the repository
```bash
git clone [https://github.com/your-username/galactic-horizons.git](https://github.com/your-username/galactic-horizons.git)
cd galactic-horizons

2. Install dependencies
Bash

npm install
# or
yarn install
# or
pnpm install

3. Run the development server
Bash

npm run dev
# or
yarn dev
# or
pnpm dev

Open http://localhost:3000 in your browser to see the landing page in action.
